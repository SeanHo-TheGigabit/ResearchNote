```bash
# 2. 创建 API 路由
# app/api/chat/route.ts
```

```typescript
import { openai } from '@ai-sdk/openai';
import { streamText } from 'ai';

export async function POST(req: Request) {
  const { messages } = await req.json();

  const result = streamText({
    model: openai('gpt-4'),
    messages,
  });

  return result.toDataStreamResponse();
}
```

> 就像盖房子，你可以用预制板快速搭建框架，然后专注于室内设计。
> 还是要从烧砖开始？"

**质疑2："AI生成的代码质量不好"**

回应：
> "这是个好问题。AI生成的代码确实不完美。
>
> 但问题是：你是要完美的代码，还是要快速验证想法？
>
> v0生成的UI，可以让你在30秒内看到效果，验证想法是否可行。
> 如果可行，你再花时间优化代码。
> 如果不可行，你只浪费了30秒，而不是3天。
>
> 而且，AI生成的代码质量在快速提升。
> Vercel的模板，都是生产级别的，有测试、有最佳实践。
>
> AI不是要替代你，是要让你更快。"

**质疑3："学了Next.js，万一过时了怎么办？"**

回应：
> "这是对的，技术会过时。但有些东西不会过时：
>
> 1. **全栈思维**：Next.js教你的是全栈开发思维，这个不会过时
> 2. **React生态**：Next.js基于React，React是最大的前端生态
> 3. **边学边赚**：学Next.js的同时，你可以做产品赚钱，这不冲突
>
> 而且，退一万步讲：
> - Next.js是目前最火的全栈框架，140万月活开发者
> - Vercel估值几十亿美元，不会轻易倒闭
> - 就算过时了,你用Next.js赚到的钱、学到的思维，不会过时
>
> 与其担心过时,不如现在就开始行动。"
